# Quick Start Guide

Get BetterFrame running in 5 minutes!

## Step 1: Generate Icons

Open `generate-icons.html` in your browser:
1. Double-click `generate-icons.html` (or open it in Chrome)
2. Click each "Download" button to save the three icon files
3. Save them in the `icons/` folder with these exact names:
   - `icon16.png`
   - `icon48.png`
   - `icon128.png`

## Step 2: Load Extension in Chrome

1. Open Chrome and go to: `chrome://extensions/`
2. Turn on **"Developer mode"** (toggle in top-right corner)
3. Click **"Load unpacked"**
4. Select the `BetterFrame` folder
5. The extension will appear in your extensions list!

## Step 3: Test It Out

## Keyboard Shortcuts

- `Shift + Left Arrow` = Skip backward 5 seconds
- `Shift + Right Arrow` = Skip forward 5 seconds

## Troubleshooting

**No icons showing up?**
- Make sure you saved all three PNG files in the `icons/` folder
- Refresh the extension on `chrome://extensions/`

**Buttons not appearing on Frame.io?**
- Open the browser console (F12) and look for "BetterFrame:" messages
- Try refreshing the Frame.io page
- Make sure you're on a review page (URL contains `/reviews/`)

**Still having issues?**
- Check the console for error messages
- See the full README.md for detailed troubleshooting
